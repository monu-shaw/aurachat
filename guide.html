Welcome to AuraChat

AuraChat runs a powerful AI entirely in your web browser. No servers, complete privacy. Follow these 3 simple steps to get started.

1

Download the AI Model

AuraChat requires an AI model file to function. You will need to download the Gemma-4-E2B LiteRT model to your computer.

File Name: gemma-4-E2B.litertlm
Approximate Size: ~1.3 GB
Download from Hugging Face
Note: Ensure you download the correct `.litertlm` file to your Downloads folder or Desktop.
2

Load the Model into AuraChat

Once the file is downloaded, you need to connect it to the application.

  • Open the AuraChat App.
  • Look at the top navigation bar and click the + Model button.
  • A file picker will open. Locate and select the gemma-4-E2B.litertlm file you just downloaded.
3

Wait for the Magic (Only Once!)

AuraChat will now save the model securely inside your browser's private storage (OPFS). You only have to do this once!

The next time you open AuraChat, it will instantly load the model from your local cache—no internet connection or re-uploading required. Look for the green Model Ready badge in the navigation bar.

💡 Pro Tips
  • Privacy First: Your chat data and the model file never leave your device. Everything runs locally via WebAssembly.
  • Performance: Ensure hardware acceleration is enabled in your browser settings for the fastest response times.
  • Storage: If you ever need to clear your browser cache, you might need to re-upload the model file.
